# Severity Matrix — NCR Classification

## Severity Levels

| Severity | Definition | Impact | Action |
|----------|-----------|--------|--------|
| CRITICAL | Safety / structural / fire / waterproofing failure | Immediate risk to life or building integrity | STOP WORK, escalate to QAQC Manager |
| MAJOR | Performance-impacting spec deviation | Building function compromised, rework required | REJECT work, corrective action mandatory |
| MINOR | Documentation gap / cosmetic / within tolerance | No performance impact, admin issue | CONDITIONAL, address before final inspection |

---

## CRITICAL Severity Triggers

- Structural integrity compromised (rebar spacing, concrete strength below design)
- Safety hazard (falling debris risk, electrical hazard, fire system failure)
- Waterproofing breach in wet areas / basement / roof
- Fire-rated assembly non-compliance (door rating, penetration sealing)
- Unauthorized work past Hold Point
- Material installed without approved MAR

**Response:** Immediate stop work, QAQC Manager escalation, root cause analysis mandatory.

---

## MAJOR Severity Triggers

- Spec deviation affecting performance (wrong material grade, incorrect thickness)
- Material substitution without approval
- Failed pressure / strength / load test
- Missing approved MS before execution start
- Work not matching approved shop drawings
- Test certificate from non-accredited lab

**Response:** Reject work, corrective action plan required, re-inspection mandatory.

---

## MINOR Severity Triggers

- Documentation incomplete (minor format issues, missing signatures)
- Cosmetic deviation within acceptable tolerance
- Administrative non-compliance (wrong form, late submission)
- Minor surface defects correctable without rework

**Response:** Note and track, address before final acceptance.

---

## Severity Decision Tree

```
Is safety or structural integrity at risk?
  → YES → CRITICAL

Does deviation affect building performance or function?
  → YES → MAJOR

Is it purely documentation, cosmetic, or within tolerance?
  → YES → MINOR

When in doubt → Escalate to Human QA Engineer
```
