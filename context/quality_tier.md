# Quality Tier — Project Quality Level Definition

## Project Quality Level

> Set per project in `context/project_context.md`

Default options:
- **STANDARD** — Normal commercial / residential
- **PREMIUM** — High-end residential / 4-star hotel
- **LUXURY** — 5-star hotel / branded residence
- **ULTRA-LUXURY** — Ultra 5-star branded / landmark project

---

## Tier Rules (Default: ULTRA-LUXURY)

### ULTRA-LUXURY (5-star branded)

```
All finishes: mockup mandatory before mass production
Marble: sample board + installed mockup + client sign-off
Paint: minimum 3 color samples, consultant + client approval
Joinery: full-size mockup, brand representative review
Wall flatness: 2mm/2m (not 3mm standard)
Tile joint: max 1.5mm (not 2mm standard)
Paint sheen: uniform, no roller marks visible at 45° raking light
Marble veining: matched per approved sample board
Surface tolerance: 1.5mm/2m straight edge (luxury threshold)
Lippage: max 0.5mm adjacent tiles
```

### PREMIUM (4-star)

```
Mockup: required for primary finishes only
Wall flatness: 3mm/2m
Tile joint: max 2mm
Paint: 2 color samples minimum
Surface tolerance: 2mm/2m
```

### STANDARD

```
Mockup: recommended, not mandatory
Wall flatness: 4mm/2m
Tile joint: max 3mm
Standard industry tolerances apply
```

---

## Mockup Approval Hierarchy (Luxury)

```
1. Material sample → Consultant approval
2. Workshop sample → Consultant + Client approval
3. Site mockup built → Brand rep + Consultant + Client review
4. Sign-off → Mass production release

RULE: No mass production without all three approvals.
Agent holds execution start until mockup sign-off logged.
```
