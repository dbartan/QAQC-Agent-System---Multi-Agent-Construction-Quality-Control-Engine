# Project Phases

## Phase Structure

| Phase Code | Phase Name | Description |
|------------|-----------|-------------|
| P1 | Earthworks & Substructure | Excavation, piling, foundation, basement, backfill |
| P2 | Shell & Core | Structural frame, concrete, rebar, formwork, core walls |
| P3 | Façade & Envelope | Curtain wall, cladding, waterproofing, glazing |
| P4 | MEP Rough-In | Mechanical, electrical, plumbing first-fix, ducting, cable trays |
| P5 | Fit-Out (Finishes) | Plastering, tiling, painting, joinery, ceilings, flooring |
| P6 | Snagging & De-snag | Defect identification, correction, re-inspection |
| P7 | Testing & Commissioning | Systems testing, balancing, performance verification |
| P8 | Handover | Final inspection, O&M manuals, as-built drawings, client handover |

## Phase-Gate Rules

- No phase begins without all prior-phase NCRs closed
- Each phase requires phase completion certificate
- Handover documentation must be 90%+ complete before Phase 7 starts
- Phase-specific mockups (Phase 5) must be approved before mass production
